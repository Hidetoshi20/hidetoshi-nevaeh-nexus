## 🟢 Wednesday Program（Deep Core & Mobility）

> 目标：重塑呼吸与腹压控制，强化腹横肌与斜腹，改善骨盆前倾与胸椎活动。

### 当日结构（Wednesday 顺序与必做）

1. Reset & Activation（必做 5 分钟）
2. Main Work：Deep Core Circuit（必做 12–15 分钟）
3. Accessory + Posture：Mobility Flow（可选 8–10 分钟）
4. Core & Conditioning：Isometric Integration（可选 6 分钟）
5. Cooldown & Log（必做 2–3 分钟）

### 1. Reset & Activation（5 分钟）

- 90/90 Breathing（90/90 呼吸）：2 分钟（脚跟搭椅，腰背贴地，呼气肋骨下沉）
- Stomach Vacuum（真空腹）：3–4 轮（10–15s/轮；四足跪或坐姿）

### 2. Main Work：Deep Core Circuit（3 轮）

- Dead Bug（死虫式 - 标准版）12 次/侧（Tempo 3-1-3；腰椎贴地）
- Side Plank Rotation（侧桥旋体）10 次/侧（眼看肘，胸椎旋转）
- Hollow Hold（空心支撑）20–30s（若腰离地 → 屈膝退阶）

质量标准与退阶：

- 若死虫腰痛 → 改 Bird-dog（鸟狗式，每次 5s 等长）
- 若侧桥旋体不稳 → 改静态侧桥 20–30s/侧

### 3. Accessory + Posture：Mobility Flow（10 分钟）

- 参考：[Home Mobility & Recovery Flow](./home-mobility-recovery-flow.md)
- 建议序列（选 2–3 个）：
  - Cat-Camel（猫牛式）→ Thread the Needle（穿针式） × 8（胸椎旋转）
  - Half-kneeling Hip Flexor w/ Reach（半跪髋屈肌伸展 + 前伸） × 6/侧（髋屈肌放松 + 臀激活）
  - Inchworm（毛毛虫步）→ Down Dog（下犬式） + Heel Pulse（跟腱弹动） × 6（腘绳/小腿/肩部）

### 4. Core & Conditioning：Isometric Integration（6 分钟，可选）

- Bear Hold（熊式支撑）呼吸 2 × 20s（抗伸展 + 呼吸整合）
- Glute Bridge Iso Hold（臀桥等长保持）2 × 40s（臀主导，保护腰椎）

### 5. Cooldown & Log（2–3 分钟）

- 拉伸：胸椎、髋屈肌、臀肌（30–45s/动作）
- 记录：真空腹秒数 → 死虫是否腰痛 → Mobility 模块完成度 → sRPE × 时间

选择逻辑（Wednesday）：

- 腰背敏感 → 优先 Hollow 退阶与 Bird-dog，Mobility 选胸椎与髋屈肌序列
- 时间紧张 → 执行 1–2–5 步（跳过 3/4 步）

## 🔵 Saturday Program（Cardio）

> 目标：通过 HIIT 或 LISS 提升能量消耗与心肺，同时保持核心与肩胛的基本稳定。

### 当日结构（Saturday 顺序与必做）

1. Reset & Activation（可选 5 分钟：90/90 2m + 轻动态）
2. Cardio Main Course（必做：HIIT 20m 或 LISS 40m）
3. Accessory + Posture：Shoulder Micro Doses（可选 6 分钟）
4. Core & Conditioning：Core Micro Doses（可选 6 分钟）
5. Cooldown & Log（必做 2–3 分钟）

### 2. Cardio Main Course（选择其一）

Option A · HIIT（20 分钟）

- Keep 关键词：`无跳跃HIIT` `哑铃燃脂`
- 结构：热身 5m → 高强度 15m（30s ON / 30s OFF）→ 拉伸
- 提示：保持骨盆中立与鼻吸口呼；若动作含跳跃 → 改为低冲击版本

Option B · LISS（40 分钟）

- Keep 关键词：`室内燃脂走` `低冲击有氧`
- 户外替代：骑行或快走
- 心率目标：120–140 BPM；呼吸节奏稳定（鼻吸口呼）

选择逻辑（Saturday）：

- 疲劳高/恢复优先 → 选 LISS；状态好/时间紧 → 选 HIIT

### 3. Accessory + Posture：Shoulder Micro Doses（6 分钟，可选）

- 弹力带 Face Pull（面拉）15 次（顶端外旋停 2s）
- 弹力带外旋（ER）12 次/侧（肘贴身 90°）
- 旅行箱行走（Suitcase Carry）30m（单手，肩胛下沉，核心抗侧屈）
- 替代：直臂下拉（弹力带）15 次（肩胛下压）可替换 Carry；左肩弱侧多做 2 次

### 4. Core & Conditioning：Core Micro Doses（6 分钟，可选）

Set A · Anti-Extension（抗伸展）

- Plank Saw（平板锯）30s
- Dead Bug（死虫式）10/侧

Set B · Anti-Rotation（抗旋）

- Pallof Press（抗旋压推）12/侧
- Shoulder Tap Plank（肩触平板）20 次

做法：任选 1 组别，做 2–3 轮；若当天已完成大量核心 → 跳过此步

### 5. Cooldown & Log（2–3 分钟）

- 拉伸：髋屈肌、腘绳、小腿、胸肌（30–45s/动作）
- 记录：心率区间 → sRPE → 主课程类型（HIIT/LISS）→ 肩颈/腰背反馈
