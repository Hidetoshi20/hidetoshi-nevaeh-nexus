## 技能自评 Skills Assessment

> _注：更新于 2025-12-03_

### 1. 技术 Technical

- **停球 (First Touch)**: **良好**。基本可以停住球，控制在舒适区。
- **传球 (Passing)**:
  - **惯用脚 (右脚)**: 会使用正脚背（长传/短传）。会一些外脚背，但不太喜欢用，**更倾向于练习逆足能力**。
  - **非惯用脚 (左脚)**: 可进行短传，但传高球/长传比较吃力。
- **盘带 (Dribbling)**: **较差**。
  - **优点**: 会抬头观察。
  - **弱点**: 面对防守缺乏技巧（仅限于简单的左拨右拨），过人成功率不稳定（看运气）。
- **射门 (Finishing)**: **发力与稳定性不足**。
  - **正脚背**: 发力尚可，但稳定性非常差（_个人倾向减少此类训练，认为现代足球更偏向内脚背/抽射_）。
  - **内脚背**: 发力不充分，吃球部位不准确，一发力容易踢歪。

## 身体素质关联 Physical Cross-Reference

_详情见 [Physical Profile](./physical-profile.md)_

- **优势**: 身高 184cm（制空/护球潜力）。
- **劣势**:
  - **骨盆前倾**: 可能会影响踢球时的摆腿幅度与腰部压力。
  - **核心/呼吸**: 对抗下的平衡能力可能受限。
  - **左侧肢体**: 左脚发力或支撑稳定性可能较弱（需重点关注）。

## 训练目标 Training Goals (Football)

1. **专项技术突破**:
   - **盘带**: 增加 1v1 突破技巧（如沉肩、变向），不仅仅是简单的拨球。
   - **射门**: 重点攻克**内脚背/抽射**的吃球部位与发力机制，提升稳定性。
   - **逆足**: 强化左脚高球与长传能力。
2. **巩固优势**: 保持现有的停球与视野观察习惯，尝试在更高对抗下运用。
3. **身体转化**: 解决骨盆前倾对摆腿的影响，将力量转化为射门/长传的爆发力。
4. **建立训练习惯**: 每周固定安排 1-2 次专项技术训练（可替代 LISS/HIIT）。
