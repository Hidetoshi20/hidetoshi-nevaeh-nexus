# 健身房训练计划（Gym Plan）

> 返回主页：[Hidetoshi Fitness & Nutrition Plan](../README.md)

> 训练安排以 [Training Goals](../training-goals.md) 的周期指标为导向，动作结构与器材策略来自 [Plan Rationale](../plan-rationale.md)。若当周无法到馆，请参考对应的 [Home Plan](../home-plan/README.md) 获得同结构的替代动作。
