# 🏠 HIIT + Core Circuit（居家 HIIT + 核心）

> 目的：在家完成 18–25 分钟的心肺 + 核心训练，覆盖 **Priority 1（核心+体脂）** 与 **Priority 2（姿态控制）**，并与周末户外骑行/LISS 互补。

## 🎯 Priority Snapshot

- **P1**：维持每日热量缺口 → 让 EMOM / Interval 的平均 RPE 落在 7 左右，并记录 sRPE × 分钟。
- **P2**：所有动作保持肋骨下压、骨盆中立；跳跃改为低冲击版本也要确保膝盖对齐。
- **P3（可选）**：若想刺激上肢/下肢肌肉，可把 DB Thruster、Swing 等动作以 4-1-2 节奏执行。

## ⏱️ Session Layout（约 20 分钟）

1. **Block A｜Reset & Warm-up（4–5 min）**
2. **Block B｜Main Conditioning（12–15 min）**：在 EMOM 或 30:15 间歇中二选一
3. **Block C｜Core Finisher（4–5 min）**
4. **Block D｜Cooldown & Log（3 min，可选）**

---

### Block A｜Reset & Warm-up

1. 90/90 呼吸 + 伸臂 (90/90 Breathing + Reach) × 4 呼吸
2. 开合跳 / 低冲击开合步 (Jumping Jack) × 40s
3. 世界上最佳拉伸 (World’s Greatest Stretch) × 6/侧
4. 平板走路 + 肩触 (Plank Walk-out + Shoulder Tap) × 6

> 若当天要做高冲击版本，追加 20s 高抬腿或原地跑作为神经激活。

---

### Block B｜Main Conditioning（选择其一）

**Option 1：EMOM 15’**

| Min | 动作（节奏建议） |
|-----|----------------|
| 1 | 哑铃推举深蹲 (DB Thruster) 12 次（3-1-2；若 <35s 完成则 +2 次或延长节奏） |
| 2 | 登山者或熊式抬手 (Mountain Climber / Bear March) 40s |
| 3 | 交替反向弓步 + 弯举 (Alternating Reverse Lunge + Curl) 12/侧（负重或加弹力带） |

**Option 2：30:15 间歇 × 4 轮（总 18 min）**

1. 30s 高抬腿行进 + 哑铃刺拳 (High-Knee March + DB Punch)（低冲击：快走 + 空中刺拳）  
2. 30s 侧向滑冰跳 (Lateral Skater Hop)（低冲击：侧步+触地）  
3. 30s 俯卧撑转下犬式 (Push-up to Down Dog)（跪姿替代）  
4. 30s 硬拉到高拉 (Deadlift to High Pull)（5–10 kg，节奏 3-1-2）  
5. 30s 开合平板 / 交替侧点 (Plank Jack / Side Tap)  
6. 45s 休息 → 深呼吸，记录心率/RPE  

> EMOM 无法在 50s 内完成 → 下次改 Option 2；若间歇心率>85% HRmax，延长休息到 60s。

---

### Block C｜Core Finisher（2 轮）

| Exercise | Time/Reps | Cue |
|----------|-----------|-----|
| 空心摇 / 空心支撑 (Hollow Rock / Hold) | 25s | 保持下背贴地，肋骨下压 |
| 侧桥臀提 (Side Plank Hip Lift) | 12/侧 | 骨盆保持水平，肩胛稳定 |
| 熊爬前后 (Bear Crawl Forward/Backward) | 3 步来回 × 2 | 膝离地 2–3 cm，核心抗旋 |
| 缓慢 V 坐至船式 (Slow V-sit to Boat Pose) | 10 | 控制离心 4 秒 |

> 想增加脂肪燃烧，可额外加入 **Farmer Carry 30m/侧 + 休 30s** 作为第 3 轮。

---

### Block D｜Cooldown & Log

- 90/90 Breathing 或 Crocodile Breathing 2 min
- 腿后侧/臀部/胸椎拉伸各 30s/侧
- 日志字段：所选 Option、完成轮数、平均 RPE/心率、核心/膝盖手感、当天累计步数或骑行数据。若 24 小时内仍感觉心率偏高，下次可减一轮或与 LISS 互换。

---

## Integration Tips

- 若当日已完成户外骑行，可只做 Block C（核心）+ Block D，用于巩固 Priority 1。
- Deload 周改成 Option 2 低冲击版本 + Block C 一轮即可。
- 与力量日间隔 ≥6 小时；若必须同日，安排在早晨/午间，把力量训练放在下午，或反之。
