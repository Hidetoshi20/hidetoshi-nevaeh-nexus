# 居家训练计划（Home Plan）

本目录提供与健身房版本目标一致的“居家替代方案”。设备假设：一对 5 kg 轻哑铃、瑜伽垫、自重训练空间；如有弹力带/门框固定点，可用于动作进阶（详见各页“进阶/退阶”）。

> 使用方法：与健身房版遵循相同周频率（Push/Pull/Lower/Recovery）；如当周只能在家训练，可直接按下列周表执行。个人档案、记录、营养与测试文档与健身房版共享。
