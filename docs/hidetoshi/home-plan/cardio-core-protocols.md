# 🏠 Cardio & Core Protocols（居家心肺 + 核心整合）

> 统一整合 HIIT、LISS 与 Core Micro，围绕 Priority 1（核心稳定 & 体脂）与 Priority 2（体态矫正），适配当前居家器材与个人情况（APT、上交叉、左肩内旋、腰椎风险）。

## 何时选哪种
- 疲劳高/腰背敏感 → 选 LISS（低冲击稳态，RPE 5–6）。
- 时间紧或想提升代谢 → 选 HIIT（18–25 分钟，含低冲击替代）。
- 力量日后补充或忙碌日维持 → 选 Core Micro（6–8 分钟）。

## LISS（30–45 分钟，RPE 5–6） {#liss}
- 模式（选 1–2）：
  - 室内快走/原地踏步 20–30'（节拍 120–130 BPM）
  - 椅子/台阶踏步 15'（2–3' 上下 + 1' 慢走 × 4 轮）
  - 室内骑行 30'（Cadence 80–90，轻中阻力）
- 姿态要点：肋骨下压、骨盆中立、肩胛下沉；每 10' 自检一次。
- 结束：90/90 呼吸 2' + 针对性拉伸各 30–40s。

## HIIT（18–25 分钟，RPE 7 左右） {#hiit}
- 结构（二选一）：
 1) EMOM 15’（每分钟开始）
    - 1：DB Thruster 12 次（3-1-2 节奏）
    - 2：登山者或熊式抬手 40s（低冲击：快步抬膝）
    - 3：交替反向弓步 + 弯举 12/侧（可加弹力带）
 2) 30:15 × 4 轮（总 18’）
    - 高抬腿行进 + DB 刺拳 → 侧向滑步 → 俯卧撑转下犬 → 硬拉到高拉 → 平板侧点 → 休 45s
- 个人化替代：膝不适 → 跳跃改侧步/踏步；左肩不适 → 俯卧撑改上斜；腰背敏感 → 高拉改弹力下拉。
- 收尾：Core Finisher（见下）+ 呼吸 2'。

## Core Micro（6–8 分钟，力量日后或忙碌日） {#core-micro}
- A：抗伸展/抗旋（各 2 轮）
  - Dead Bug 10/侧（腰背贴地，呼气伸展）
  - Half-kneeling Pallof 12/侧（门锚/毛巾等长）
  - Forearm Plank + 前伸 30s（抗旋）
- B：髋稳定/侧向（各 2 轮）
  - Side Plank Clamshell 12/侧（骨盆水平）
  - Fire Hydrant + Kickback 12/侧（末端停 1s）
  - Reverse Crunch 12（3-1-3）

## Keep App 课程匹配指南 {#keep-app-course-matching-guide}

### HIIT (18-25分钟)
**关键词：** HIIT燃脂、哑铃HIIT、无跳跃HIIT | **课程：** 燃脂HIIT系列、哑铃HIIT训练 | **调整：** 膝不适→无跳跃版本；肩不适→上斜俯卧撑

### LISS (30-45分钟)
**关键词：** 低强度有氧、室内骑行、燃脂走 | **课程：** 低冲击燃脂训练、有氧操初级 | **要点：** 保持骨盆中立，选择呼吸指导课程

### Core Micro (6-8分钟)
**关键词：** 核心训练、死虫式、平板支撑 | **课程：** 死虫式训练、弹力带核心训练 | **重点：** Dead Bug→死虫式；侧平板→侧平板变式

**搜索策略：** 居家训练+初级/中级+对应时长+哑铃/弹力带
**收藏夹：** HIIT燃脂、低冲击有氧、核心稳定、个人化替代
