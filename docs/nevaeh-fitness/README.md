## 💡 快速指引 Quick Guide

1.  **新手建议**：先从 [Home Workouts](./home-workouts/README.md) 开始，建立动作模式与运动习惯。
2.  **进阶路线**：当居家训练感觉轻松，或希望进一步改善体型（增肌/增重）时，尝试 [Gym Workouts](./gym-workouts/README.md)。
3.  **核心原则**：始终坚持 [Correction First（矫正优先）](./training-methodology.md#core-principles)，不要为了重量牺牲姿态。
