## 💡 训练提示

1.  **动作质量**：每个动作都要确保姿态正确，尤其是脊柱的中立位。
2.  **呼吸**：不要憋气，发力时呼气，还原时吸气。
3.  **循序渐进**：如果觉得轻松，可以增加弹力带的阻力或增加每组次数。
