# 🏋️ Gym Workouts

> **适用场景**：当你有机会前往健身房，使用固定器械与自由重量进行更系统的训练时。
> **核心目标**：利用更稳定的器械轨迹，安全地增加负重，促进肌肉生长与力量提升。

---

## 训练计划清单

目前处于**基础适应期**，建议采用全身训练模式（Full Body），每周 2-3 次，每次间隔至少 48 小时。

- [📋 基础全身训练 (Foundation Full Body)](./full-body-foundation.md)
  - _适合初次进入健身房，建立动作模式，熟悉器械。_

---

## 健身房训练提示

1.  **安全第一**：不熟悉的器械请先阅读说明或咨询巡场教练，不要勉强上大重量。
2.  **卫生礼仪**：使用完器械后擦拭汗渍，将哑铃/杠铃片归位。
3.  **循序渐进**：前几次训练以"找感觉"为主，重量选择能轻松完成 15 次的级别。
