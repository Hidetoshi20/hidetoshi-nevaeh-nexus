# 🍽️ 饮食偏好

我偏好新疆风味与清真饮食，强调“重香料、重层次、重原味”。下面是更易实践的偏好与原则，帮你快速做出符合口味的菜。

## 我的原则

- 不使用猪肉及其制品
- 肉类以羊肉、牛肉为主，蔬菜以土豆、洋葱、胡萝卜为常用搭档
- 香料要鲜明但不刺鼻，突出孜然、花椒、八角与干辣椒的层次
- 口味偏重但不过盐，追求回味与层次

## 常备香料与调味

- 孜然：炖、焖、烤通用；与洋葱搭配提升肉香
- 花椒：提供麻香，与孜然搭配形成新疆经典风味
- 八角：炖菜提层次；与胡萝卜、牛羊肉相性好
- 干辣椒：增辣提香，适量使用不刺喉
- 小茴香：炖肉增香，适合抓饭、焖面
- 肉桂：暖香增加厚度，少量即可
- 洋葱、蒜：打底增甜增香，几乎是必备

## 常用食材与搭配

- 羊肉/牛肉 + 洋葱 + 孜然：硬菜基础风味
- 牛肉 + 土豆 + 八角：浓郁炖菜经典组合
- 面食（面条、饼子、包子） + 羊/牛肉：主食与硬菜的平衡
- 胡萝卜 + 抓饭：提升甜香与色泽

## 常用烹饪方式

- 炖：浓郁、层次深（如牛肉炖土豆）
- 焖：保留原味与口感（如焖面、焖饼子）
- 炒：快手菜，锁住鲜嫩（如芹菜肉沫）
- 烤：突出孜然香与肉香（如烤串风格）
- 蒸：面食类（包子、饼）

## 风味搭配建议

- 孜然 + 花椒：新疆标志性组合
- 八角 + 肉桂：炖菜的温暖底香
- 胡萝卜 + 牛羊肉：甜香与肉香平衡
- 洋葱 + 干辣椒：香甜中带热度

## 文化与习惯

- 清真饮食：不使用猪肉及其制品
- 游牧文化：以牛羊肉为主，讲究耐饱与能量
- 丝绸之路：融合多地香料与做法，形成独特层次
