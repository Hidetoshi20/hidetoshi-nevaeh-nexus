# 📚 烹饪技巧

更注重“能做成”和“风味可控”的实用技巧，结合新疆偏好给出建议与排错。

## 基础技巧
- 火候控制：小火（保温/焖）、中火（常规翻炒）、大火（收汁/爆香）
- 调味节奏：盐最后放，避免过早出水影响口感
- 切配规范：主材块状统一，小料切碎；保证受热与口感一致
- 底味打底：洋葱/蒜先爆香，可提升整体层次

## 新疆风味要点
- 香料主次：以孜然为主香，八角/花椒为厚度，干辣椒点缀辣度
- 肉类处理：牛羊肉先煸香再炖，表面微焦更增香
- 组合偏好：牛肉+土豆、羊肉+洋葱、胡萝卜+抓饭
- 焖与炖：焖更保留原味与口感，炖追求软烂与浓郁

## 常见问题与排查
- 肉不软：延长小火炖煮时间或切更小块；增加液体覆盖
- 汤太淡：适当加盐或少量酱油，并增加收汁时间
- 香料抢味：减少八角/花椒用量，保留孜然为主香
- 过辣/过麻：减少干辣椒/花椒，用洋葱与胡萝卜平衡

## 推荐搭配
- 孜然+花椒：新疆标志性组合，适合炒与烤
- 八角+肉桂：炖菜暖香底味，少量使用
- 洋葱+干辣椒：爆香增甜并带热度
